HTML
Hypertext Marker Language è un linguaggio di markup usato per creare siti web.
MARKER
Il marker è un tag, scritto tra parentesi angolari(<,>) usato per dare un comando al broswer che legge il documento html.
ATTRIBUTO
Gli attributi sono specificazioni del tag come "src" per le immagini.
COLLEGAMENTI IPERTESTUALI
I collegamenti ipertestuali sono ollegamenti tra pagine web ed altre o tra documenti e pagine web.
CSS
Cascade Style Sheet è un linguaggio usato per definire la formattazione di struttrura di HTML, XHTML e XML.
BODY
Il body o corpo in italiano è tutto cio che viene visto quando una pagina html viene caricata. Ogni elemento fuori da body non si vede.
FONT
Il font è il carattere es:Serif, Times New Roman, Sans-Serif, Arial.
LAYOUT
La disposizione sulla pagina degli elementi che costituiscono la pagina web.
SERIF
Le serif, in italiano "grazie" sono una famiglia di più caratteri che hanno dei prolungamenti ai vertici delle lettere che servono a decorare le stesse.
LINGUAGGIO DI MARKUP
Un linguaggio di markup è un insieme di comandi che hanno come obbiettivo definire e descrivere la struttura di una pagina.
FORMATO DI STRUTTURA
Formato di Struttura è il modo in cui si vede ciò che viene scritto.
MODULI
I moduli sono campi di scrittura o altro in cui è possibile inserire testo o scelte tra più opzioni.
INTESTAZIONI E PIÈ DI PAGINA
Le intestazioni e i piè di pagina sono piccole note in cima o in fondo alla pagina.
LISTE PUNTATE
Liste di frasi o parole in cui ognuna all' inizio ha un indicatore come un numero, una lettera, un cerchio o un quadrato.
JPEG
JPEG è l'acronimo di Joint Photographic Experts Groupn, è il primo standard internazionale di compressione dell'immagine digitale.
SERVER WEB
Un server web è un computer molto potente che "ospita" il sito web che si sta cercando.
DNS
Un DNS (Domain Name System) è un server che, come farebbe un centralino telefonico, dice all' utente sulla quale si trova il sito cercato.
INDIRIZZO IP
L' indirizzo ip è come un numero di telefono che identifica il dispositivo da cui è iniziata la ricerca.
EDITOR
Un editor è un programma che consente di scrivere in Html. Anche il blocco note è un editor.
BROSWER
Un broswer è un applicazione che consente di navigare tra i siti web messi a disposizione sul World Wide Web.
