le proprietà css

Le proprietà delle regole CSS indicano qual è l'elemento del selettore che si vuole modificare. Queste possono modificare testo, colore, allineamenti margini e molto altro. Di seguito un elenco delle proprietà diviso in categorie.

  1. COLORI
  2. FONT
  3. TESTO
  4. I BOX
  5. LISTE, TABELLE E MODULI
  6. LAYOUT
  7. IMMAGINI
